# Here's a list of external grains currently available

[db](https://github.com/viggou/praia-db) - PostgreSQL and mariadb/mysql driver

[redis](https://github.com/viggou/praia-redis) - Redis client

[dotenv](https://github.com/viggou/praia-dotenv) - .env file loader

[arp](https://github.com/viggou/praia-arp) - Cross-platform ARP table access

[prscan](https://github.com/viggou/prscan) - Network scanner (also functions as a CLI tool)

## Grains with C++ plugins
[archive](https://github.com/viggou/praia-archive) - tar.gz and zip support

[curses](https://github.com/praia-lang/curses) - curses library for Praia

[pcap](https://github.com/viggou/praia-pcap) - libpcap packet capture library

[wmon](https://github.com/viggou/praia-wmon) - WiFi monitor mode and 802.11 frame parsing library

[l2](https://github.com/viggou/praia-l2) - Layer 2 raw ethernet frame access (AF_PACKET/BPF)

[ble](https://github.com/viggou/praia-ble) - BLE scanning and GATT library
